# About this website

This website serves as study notes of [Dr. Chanh Minh Tran](https://chanh1964.github.io/)'s research on k-anonymity privacy preserving models. It also provides (re)implementations of several k-anonymity algorithms for research and fact-checking purposes.

Currently, all content on this website only reflect Dr. Chanh's own investigation, understanding, opinion, etc. 

In the future, it is expected that the content this website will be used (in whole or in part) as an official output of the [SDC4Society project](https://www.sdc4society.org/).
