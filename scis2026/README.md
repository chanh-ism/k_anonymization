This branch provides the scripts for reproducing the experimental results of the work ___"Evaluation of Data Utility, Classification Consistency, and Vulnerability of k-Anonymization Algorithms"___, presented at ___[SCIS 2026](https://www.iwsec.org/scis/2026/program.html?20260126#3C1)___.

Run the provided ipython notebooks in the following order:
1. `scis2026_1_create_sample.ipynb`
2. `scis2026_2_evaluate.ipynb`
3. `scis2026_3_plots.ipynb`

After that, the results are stored in `./results/`.
